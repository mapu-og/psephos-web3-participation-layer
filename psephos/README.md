# Psephos — On-Chain Surveys, Polls, and Votes

This project implements a simple on-chain participation layer where creators can fund surveys, polls, or votes with ETH and participants earn rewards for submitting responses. The product code lives under the `psephos/` directory.

If you are reviewing this repository for the Alchemy University certification, use the root `README.md` as the main entry point. This file provides a short product overview and instructions.

For development notes and agent prompts used during the course, see the `private_docs/` folder (not public).
